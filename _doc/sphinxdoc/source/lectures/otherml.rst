
Quelques autres problèmes de machine learning
=============================================

Le machine learning est le plus simplement un problème
d'optimisation qui minimise une erreur sur un jeu de données.
A partir de là, il n'existe pas de limites à l'imagination.
Certains problèmes autres que la régression et la classification
ont émergé car récurrents.
